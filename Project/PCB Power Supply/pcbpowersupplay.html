PCB Power Supplay DC OutPut 12v|9v|5v

Project Description

A power supply circuit design with fixed output voltages of 12V, 9V, and 5V.The circuit operates by converting a 12V AC input into a stable DC output,The goal of this design is to achieve high efficiency while minimizing component size and providing a stable output suitable for various applications.

    Components:

  • Bridge Rectifier 2A (KBP210)
  • Capacitor 1000UF 25v
  • Ceramic Capacitor 10nF 50v
  • Voltage Regulator 12v (L7812CV)
  • Voltage Regulator 09v (L7809CV)
  • Voltage Regulator 05v (L7805CV)
  • Terminal Block (for input)
  • Terminal Block (for output)
