Project Description

In hazardous environments, human safety can be at stake, and this is where technology comes to the rescue.

The robot is designed to explore hazardous areas and collect vital data such as temperature, humidity, gas presence, light intensity, and fire detection.

The goal of the robot is to enhance user safety by exploring areas that are difficult or impossible to access manually.

Robot Control System via Android Application The robot is controlled through an Android application that connects to the robot via Bluetooth, allowing the user to maneuver the robot in all directions smoothly. The application also displays all the data sent by the robot in real-time, such as temperature, humidity, gas presence, fire detection, and light levels.The app compares this data in real-time with predefined benchmarks to analyze the environment around the robot and provide accurate results regarding the danger level of the area. These analyses help the user evaluate the environmental situation instantly and make informed decisions based on potential risks the robot might encounter while exploring hazardous areas.

    Components:

  • Arduino uno R3
  • Motor driver L982N
  • Two DC Motors 6v
  • Bluetooth module HC-05
  • Body
  • Two Battery 3.7v
  • Battery holder
  • Adapter 9v
  • Flame sensor
  • LDR module
  • Ultra sonic
  • DHT11 sensor
  • MQ-4 gas sensor
